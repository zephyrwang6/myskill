# 写作质量检查清单

## 内容质量检查

### 主题明确性
- [ ] 文章有明确的主题和核心观点
- [ ] 主题在开头就有所体现
- [ ] 全文围绕主题展开，没有偏离
- [ ] 标题准确反映文章内容

### 价值主张
- [ ] 读者读完后能获得具体收获
- [ ] 提供了新的信息或独特视角
- [ ] 解决了读者的某个问题或困惑
- [ ] 内容对目标读者有实际帮助

### 逻辑结构
- [ ] 开头引入自然，能吸引注意
- [ ] 主体段落逻辑清晰，层层递进
- [ ] 段落之间过渡自然
- [ ] 结尾有力，呼应主题
- [ ] 整体结构完整（引入-展开-总结）

## 表达效果检查

### 语言风格
- [ ] 语气符合目标读者和平台调性
- [ ] 用词准确，没有歧义
- [ ] 句子长短适中，读起来流畅
- [ ] 避免了过于专业的术语（除非必要）
- [ ] 个人特色鲜明，有辨识度

### 情感共鸣
- [ ] 能够引发读者的情感反应
- [ ] 使用了生动的细节和描述
- [ ] 有真实的个人经历或感受
- [ ] 语气真诚，不矫揉造作
- [ ] 能够让读者产生代入感

### 可读性
- [ ] 段落长度适中，手机阅读友好
- [ ] 重要观点突出显示
- [ ] 使用了合适的格式（列表、引用等）
- [ ] 没有过长的句子和复杂结构
- [ ] 标题和小标题层次清晰

## 社交媒体适配检查

### 平台特性
- [ ] 符合平台的内容调性
- [ ] 长度适合平台（如微博140字，公众号可长文）
- [ ] 包含合适的标签或话题
- [ ] 有互动引导（点赞、评论、转发）
- [ ] 考虑了配图需求

### 传播性
- [ ] 标题有吸引力，愿意点击
- [ ] 有金句或爆点，容易记住
- [ ] 内容有争议性或话题性
- [ ] 适合在朋友圈或社交媒体分享
- [ ] 有二次创作的空间

### 互动设计
- [ ] 结尾有明确的互动引导
- [ ] 提出了问题让读者思考
- [ ] 鼓励读者分享自己的经历
- [ ] 有行动召唤（CTA）
- [ ] 为后续交流留下空间

## 技术细节检查

### 事实准确性
- [ ] 引用的数据有来源标注
- [ ] 专业名词使用准确
- [ ] 没有事实性错误
- [ ] 例子和案例真实可信
- [ ] 时间、地点、人物等细节准确

### 格式规范
- [ ] 标点符号使用正确
- [ ] 数字用法统一
- [ ] 专有名词大小写规范
- [ ] 引用格式正确
- [ ] 链接有效可点击

### SEO优化（如适用）
- [ ] 标题包含关键词
- [ ] 文章中关键词密度适中
- [ ] 有合适的内部链接
- [ ] 图片有ALT描述
- [ ] Meta描述吸引点击

## 不同类型文章的专项检查

### 经验分享类
- [ ] 经历真实具体，有细节
- [ ] 过程描述清晰，有起承转合
- [ ] 有明确的收获或教训
- [ ] 对他人有借鉴意义
- [ ] 情感真挚，不夸张

### 干货教程类
- [ ] 步骤清晰可操作
- [ ] 有具体的例子或演示
- [ ] 预见了可能的误区
- [ ] 提供了检查标准或结果
- [ ] 有后续的进阶建议

### 观点评论类
- [ ] 观点明确有立场
- [ ] 论证充分有理据
- [ ] 考虑了反方观点
- [ ] 逻辑严密不自相矛盾
- [ ] 结论有说服力

### 产品测评类
- [ ] 评测标准客观公正
- [ ] 使用场景描述详细
- [ ] 优缺点分析全面
- [ ] 有对比参照
- [ ] 给出了明确的购买建议

## 用户体验检查

### 阅读体验
- [ ] 开头3秒内抓住注意力
- [ ] 中间部分不会让读者走神
- [ ] 结尾给读者满足感
- [ ] 全文阅读时间预期合理
- [ ] 信息密度适中，不会疲劳

### 价值感知
- [ ] 读者觉得"学到东西了"
- [ ] 内容超出读者预期
- [ ] 有想要收藏或转发的冲动
- [ ] 愿意关注作者获取更多
- [ ] 感觉内容值得花时间阅读

### 情感满足
- [ ] 读者情绪得到宣泄或安慰
- [ ] 获得了启发或灵感
- [ ] 感觉被理解或被认同
- [ ] 产生了行动的欲望
- [ ] 对作者产生好感

## 发布前最终检查

### 内容审查
- [ ] 敏感内容审查完成
- [ ] 版权问题确认无误
- [ ] 个人隐私保护到位
- [ ] 商业利益声明清晰
- [ ] 法律合规检查通过

### 技术检查
- [ ] 链接全部有效
- [ ] 图片正常显示
- [ ] 格式在不同设备正常
- [ ] 字体和颜色搭配合适
- [ ] 加载速度可接受

### 定稿确认
- [ ] 标题最终确定
- [ ] 作者信息完整
- [ ] 发布时间确定
- [ ] 标签和分类正确
- [ ] 推广计划就绪

## 使用说明

1. **写作中检查**：在写作过程中可以随时对照检查，确保方向正确
2. **修改阶段**：完成初稿后，按照清单逐项检查和修改
3. **发布前检查**：最终发布前，用清单做最后的全面检查
4. **他人协作**：可以请他人使用这份清单给文章提意见
5. **持续改进**：根据读者的反馈，不断优化检查标准

**建议**：不要追求所有项目都完美，根据文章类型和目标，重点检查相关项目即可。

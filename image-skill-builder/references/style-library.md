## 通用配图风格

### S01: 简约手绘 (Minimalist Hand-drawn)

**一句话**: 黑色线条白底，手绘质感逻辑图
**适合**: 概念解释、流程说明、技术文章
**比例**: 16:9

```
A clean minimalist hand-drawn style illustration on a pure white background. Black ink line art with slight hand-drawn texture. No photography, no gradients. Simple geometric shapes connected by clean lines to form a logic diagram. Minimal text, only 1-3 keywords. Aspect ratio 16:9.
```

### S02: 建筑蓝图编辑风 (Blueprint Editorial)

**一句话**: 精致极简，高端技术期刊感
**适合**: 系统架构、技术分析、专业内容
**比例**: 16:9

```
A clean, refined, minimalist editorial-style diagram inspired by architectural blueprints and premium tech journals. Pure white background with subtle light gray grid lines. Precise thin lines in dark charcoal (#333333) with accent color highlights. Professional typography, geometric precision. Intellectual elegance. Aspect ratio 16:9.
```

### S03: 扁平矢量插画 (Flat Vector Illustration)

**一句话**: 明快配色，几何化人物，现代设计感
**适合**: 产品介绍、商业文章、教育内容
**比例**: 16:9

```
Modern flat vector illustration style. Clean geometric shapes, bold solid colors, minimal shading. Simplified character design with clear silhouettes. Bright, contemporary color palette. White background. Infographic-like clarity. Aspect ratio 16:9.
```

### S04: 3D 等距风 (Isometric 3D)

**一句话**: 等距透视，积木质感，科技感强
**适合**: 产品功能展示、系统说明、流程图
**比例**: 16:9

```
Clean isometric 3D illustration style. Soft shadows, rounded edges, pastel colors. Objects arranged in isometric grid perspective. Minimalist and modern. White or light gradient background. No realistic textures. Toy-like, friendly aesthetic. Aspect ratio 16:9.
```

### S05: 涂鸦笔记风 (Sketchnote / Doodle)

**一句话**: 手绘涂鸦+文字排版，信息图风格
**适合**: 会议笔记可视化、知识总结、教程
**比例**: 16:9

```
Sketchnote doodle style illustration on white background. Hand-drawn icons, arrows, and simple figures. Mixed with hand-lettered text labels. Colorful markers (2-3 accent colors) on white. Playful, educational, and easy to scan. Visual note-taking aesthetic. Aspect ratio 16:9.
```

## 通用附加指令

### 中文文字指令（需要中文时追加）
```
ALL text, labels, and annotations in the image MUST be primarily in Chinese (Simplified Chinese, 中文), using a clean sans-serif Chinese font. Technical terms and abbreviations (e.g. AI, API, LLM, RAG) may remain in English. All other text must be Chinese.
```

### 英文文字指令（需要英文时追加）
```
ALL text in the image should be in English with clean modern typography.
```

### 无文字指令（纯视觉时追加）
```
No text, no labels, no annotations in the image. Pure visual illustration only.
```

### 风格一致性锚定（批量生成时每张追加）
```
IMPORTANT: This is image [N] of [TOTAL] in a batch. Maintain exact same art style, color palette, and visual consistency with all other images in this batch.
```

### 角色一致性锚定（有角色时每张追加）
```
IMPORTANT: This is panel [N] of [TOTAL]. Same character must look identical across all panels - same hairstyle, clothing, body proportions, and facial features.
```

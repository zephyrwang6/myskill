# AI写作助手提示词模板

本文档包含各种写作场景的现成提示词模板，可根据需要直接使用或修改。

## 对话式创作模板

### 采访框架模板
```
我想写一篇关于【请填写主题】的文章，但还没想清楚怎么写。

请你扮演一个资深编辑，通过提问帮我梳理思路。规则如下：
1. 一次只问一个问题
2. 根据我的回答追问细节
3. 问题要能挖掘我的真实经历和独特观点
4. 大概问 8-10 个问题
5. 问完后帮我整理成文章大纲

现在开始第一个问题。
```

### 内容整合模板
```
根据我们的对话，请帮我：
1. 整理成一篇文章的初稿
2. 保留我的原话和表达风格
3. 补充必要的过渡段落
4. 标注哪些地方需要我补充细节
5. 给出3个可能的标题选项
```

## 大纲式创作模板

### 详细大纲模板
```
## 文章大纲：【填写文章标题】

### 一、开头：引人入胜的开场（约200字）
- 核心观点：[明确表达文章的核心论点]
- 预期反应：[预判读者可能的质疑或反对]
- 价值承诺：[告诉读者读完文章能获得什么]
- 开场方式：[故事/数据/问题/金句]

### 二、背景介绍：为什么这个话题重要（约300字）
- 现状描述：[当前的情况或问题]
- 数据支撑：[相关数据或事实]
- 痛点分析：[目标读者的真实困扰]
- 个人关联：[为什么这个问题与作者相关]

### 三、核心论点1：【论点标题】（约500字）
- 观点阐述：[详细解释这个分论点]
- 案例支撑：[具体的例子或故事]
- 方法论：[如果适用，提供具体方法]
- 注意事项：[可能的误区或提醒]

### 四、核心论点2：【论点标题】（约500字）
- 观点阐述：[详细解释这个分论点]
- 对比分析：[与其他观点的比较]
- 实践案例：[实际应用的例子]
- 适用场景：[什么情况下这个观点最有效]

### 五、核心论点3：【论点标题】（约500字）
- 观点阐述：[详细解释这个分论点]
- 常见误区：[人们常犯的错误]
- 正确做法：[应该怎么做]
- 成功案例：[成功的例子]

### 六、实操建议：具体行动指南（约400字）
- 步骤一：[具体的行动步骤]
- 步骤二：[具体的行动步骤]
- 步骤三：[具体的行动步骤]
- 工具推荐：[如果有，推荐相关工具]

### 七、总结：强化核心价值（约200字）
- 观点回顾：[简要重申主要观点]
- 行动呼吁：[鼓励读者采取什么行动]
- 未来展望：[这个话题的未来发展]
- 金句收尾：[一句有力的总结语]

### 八、互动引导
- 思考问题：[引导读者思考的问题]
- 分享邀请：[鼓励读者分享自己的经历]
- 持续关注：[如何获取更多相关内容]
```

### AI扩写模板
```
请根据以下大纲扩写文章：

[粘贴大纲]

要求：
1. 严格按照大纲结构，每个部分都要充分展开
2. 每个观点都要有具体论证，避免空泛
3. 加入生动的例子和个人色彩，让文章更有温度
4. 语气：专业但不学究，像在和朋友聊天
5. 总字数：约2500字
6. 在【】标注的地方，请重点展开描述
7. 确保段落之间有自然的过渡
```

## 补充式创作模板

### 段落草稿模板
```
## 文章草稿（请补充完整）

【开头段落】
【写1-2句话，设定文章基调和主题】。[请补充：用具体的场景、故事或数据吸引读者注意力，约100字]

【过渡段落】
这个话题之所以重要，是因为...[请补充：解释为什么这个主题值得关注，有什么普遍性，约100字]

【第一个核心内容】
首先，我想分享的是...[请补充：第一个主要观点或经历，包括具体细节和感受，约200字]

【第二个核心内容】
让我印象最深的是...[请补充：第二个重要内容，可以是一个转折或深化，约200字]

【第三个核心内容】
经过这些，我最大的收获是...[请补充：总结性的观点或感悟，约150字]

【实用建议】
如果你也面临类似情况，我的建议是：[请补充：3-5条具体可行的建议，每条50字左右]

【结尾段落】
总的来说...[请补充：呼应开头，给出有启发性的结论，约100字]
```

### 补充要求模板
```
请根据我的草稿，补充 [请补充] 标注的部分。

要求：
1. 保持我的叙述风格和语气
2. 内容要具体生动，避免空泛
3. 补充的内容要和我的开头自然衔接
4. 加入适当的细节和情感描述
5. 确保逻辑连贯，过渡自然
6. 如果是故事，要有起承转合
7. 如果是观点，要有说服力
```

## 改写优化模板

### 内容诊断模板
```
从现在起，你是一个冷静刻薄的「内容主编」，禁止恭维和客套，不用迎合我的观点，要站在比我更外部的视角审稿。

请你专门按照这8个维度挑毛病：
1. **思考深度**：有没有「活人」的思考过程，还是在说正确的废话
2. **故事节奏**：文章节奏是否有起伏，能吸引人读下去
3. **痛点共鸣**：有没有让读者共鸣的真实痛点
4. **人设一致性**：是否始终围绕【填写目标人设】这个人设
5. **信息价值**：读者有没有明确的信息增益
6. **案例质量**：案例是否足够好玩，有记忆点
7. **受众匹配**：目标受众是谁，范围是否合适
8. **互动性**：语气是否像在跟读者对话、能激发评论

对每一点都请给出：
- 具体问题（指出哪里有问题）
- 修改建议（应该怎么改）
- 示例（如果可能，给一个修改后的例子）

这是我写的初稿：
[粘贴文章]
```

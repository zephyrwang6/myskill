# Logo 风格指南与提示词模板

## 主流 Logo 风格

提供以下 7 种主流风格供用户选择（至少展示 5 种）：

### 风格 1：极简扁平 (Flat Minimalist)

**特征**: 几何形状、纯色填充、无渐变无阴影、现代感强
**适用**: 科技公司、SaaS 产品、互联网品牌、个人 IP

**提示词模板**:
```
A flat minimalist logo design on a pure white background. Simple geometric shapes with solid colors. No gradients, no shadows, no 3D effects. Clean vector-style illustration. The logo represents [品牌描述]. Primary color: [主色]. The design should be instantly recognizable at small sizes. Modern, tech-forward aesthetic. Square format 1:1.
```

### 风格 2：线条艺术 (Line Art / Monoline)

**特征**: 单线条绘制、等粗线条、优雅精致、可单色使用
**适用**: 工作室、设计品牌、咖啡馆、高端个人品牌

**提示词模板**:
```
A monoline logo design on a pure white background. Single continuous line weight throughout. Elegant and refined line art illustration. The logo represents [品牌描述]. Black lines on white, suitable for single-color reproduction. Sophisticated and timeless feel. Square format 1:1.
```

### 风格 3：字母标志 (Lettermark / Monogram)

**特征**: 以品牌名首字母为核心、排版驱动、识别度高
**适用**: 大型企业、个人品牌、正式机构

**提示词模板**:
```
A lettermark monogram logo featuring the letter(s) "[字母]" on a pure white background. Typographic-driven design with creative letter arrangement. The letters should be stylized and unique, forming a cohesive symbol. Clean, professional. Color: [主色]. Modern sans-serif base with custom modifications. Square format 1:1.
```

### 风格 4：徽章/印章风 (Badge / Emblem)

**特征**: 圆形或盾形边框、复古质感、内含图标和文字、有仪式感
**适用**: 社区品牌、播客、咖啡品牌、教育机构、运动品牌

**提示词模板**:
```
A badge/emblem style logo on a pure white background. Circular or shield-shaped border containing [品牌元素] icon and the text "[品牌名]". Vintage-inspired but clean execution. Balanced composition with decorative border elements. Color: [主色]. Looks great as a stamp or seal. Square format 1:1.
```

### 风格 5：吉祥物/图标 (Mascot / Icon)

**特征**: 可爱角色或抽象图标、记忆点强、适合社交媒体头像
**适用**: 游戏、教育、社交媒体、儿童品牌、IP 孵化

**提示词模板**:
```
A mascot/icon logo on a pure white background. A friendly, memorable [吉祥物描述] character designed as a brand icon. Simple enough to work at small sizes, detailed enough to be charming. [风格修饰: cute/bold/geometric]. Color palette: [配色]. Suitable for social media avatar. Square format 1:1.
```

### 风格 6：渐变现代 (Gradient Modern)

**特征**: 丰富渐变色、3D 质感、科技感强、视觉冲击力大
**适用**: AI 产品、金融科技、移动 App、创新型初创企业

**提示词模板**:
```
A modern gradient logo on a pure white background. Smooth color gradient from [色1] to [色2], creating a vibrant, dynamic feel. Abstract geometric or organic shape representing [品牌概念]. Subtle 3D depth effect. Futuristic and innovative aesthetic. High visual impact. Square format 1:1.
```

### 风格 7：手写/书法 (Handwritten / Calligraphic)

**特征**: 手写字体、自然流畅、温暖有温度、个人风格突出
**适用**: 个人博客、手作品牌、餐饮品牌、艺术工作室

**提示词模板**:
```
A handwritten calligraphic logo on a pure white background. Flowing, natural brush strokes spelling "[品牌名]". Warm, personal, and authentic feel. The lettering has [粗犷/优雅/活泼] character. Single color: [主色]. Organic and human touch. Square format 1:1.
```

## 通用提示词规则

### 基础指令（每个提示词必须包含）
```
Professional logo design. Pure white background. High contrast. Scalable vector-style quality. No watermark, no mockup, no background texture.
```

### 中文文字指令（当 Logo 需要中文时）
```
ALL text in the logo MUST use Chinese characters (Simplified Chinese, 中文) with clean, professional Chinese typography. Technical terms and brand abbreviations (e.g. AI, IoT) may remain in English.
```

### 批量变体策略

为同一品牌生成 10 个 Logo 变体时的策略：

1. **变体 1-2**: 用户选择的主风格，不同构图（横版/方版）
2. **变体 3-4**: 主风格 + 不同配色方案
3. **变体 5-6**: 主风格 + 不同图标/符号变化
4. **变体 7-8**: 相近风格的交叉尝试（如极简+线条混合）
5. **变体 9-10**: 用户指定的额外探索方向

### 图片分析模式提示词

当用户上传参考图片时，使用以下分析框架：

```
请分析这张图片，提取以下信息：
1. 图形元素：使用了什么形状/图标/符号？
2. 色彩方案：主色、辅色、背景色
3. 排版风格：字体类型（衬线/无衬线/手写）、排列方式
4. 整体风格：最接近上述 7 种风格中的哪一种？
5. 情感调性：传递什么感觉（专业/活泼/温暖/科技...）
```

# 选题策略参考

## 四大选题策略

### 1. 延伸策略（Extension）

在原观点基础上向外延展，提供更多维度、更深层次的思考。

**适用场景：**
- 原内容信息量大但未充分展开
- 有可挖掘的细节或案例
- 可补充实操方法或工具

**推文示例：**
```
原文：AI工具改变了程序员的工作方式
延伸：除了提效，AI编程工具正在重新定义"高级程序员"的标准。
会用工具 ≠ 懂原理。2026年，技术深度 + AI协作能力才是核心竞争力。
你的团队准备好了吗？
```

**公众号大纲示例：**
```
标题：《AI时代，什么样的程序员更值钱？》
1. AI编程工具的三个发展阶段
2. 传统技术能力的价值重估
3. 新型程序员的四项核心能力
4. 如何建立AI时代的技术护城河
5. 案例：三位程序员的转型故事
```

---

### 2. 反驳策略（Counter）

提出相反或不同角度的观点，引发思辨和讨论。

**适用场景：**
- 原观点存在片面性或绝对化表述
- 可提供反例或数据支撑反向论证
- 话题有争议性，适合引发讨论

**推文示例：**
```
原文：AI让内容创作变得毫无门槛
反驳：真相恰恰相反。AI降低的只是"生产门槛"，却提高了"脱颖而出门槛"。
当人人都能用AI写文章，优质内容的稀缺性反而更高。
未来比拼的是：洞察 > 工具。
```

**公众号大纲示例：**
```
标题：《AI内容泛滥时代，为什么好内容更稀缺了？》
1. 被忽视的真相：门槛的双重性
2. 数据：AI内容占比 vs 爆款内容来源分析
3. 三个被AI拉大的差距
   - 认知差距
   - 审美差距
   - 判断力差距
4. 如何在AI时代做出不可替代的内容
5. 反思：工具 vs 洞察，谁更重要？
```

### 3. 扩充策略（Expand）

对原内容的某个点进行深度展开，提供完整的解决方案或系统性知识。

**适用场景：**
- 原内容提到了有价值的点但未详细说明
- 读者对某个话题有强烈的"如何做"需求
- 可提供具体方法论、工具或案例

**推文示例：**
```
原文：Gemini 3.0的提示词技巧很强大
扩充：测试了Gemini 3.0的10个提示词模板，发现3个关键规律：
1. 角色定位要"不舒服"（反常识设定）
2. 约束条件要"极端"（逼出创造力）
3. 输出格式要"结构化"（提高可用性）
明天发完整测试报告。
```

**公众号大纲示例：**
```
标题：《Gemini 3.0深度测评：10个提示词模板 + 3大底层规律》
1. 为什么Gemini需要"特殊"的提示词
2. 10个高效提示词模板实测
   - 商业分析类（3个）
   - 内容创作类（4个）
   - 技术开发类（3个）
3. 三大底层规律详解
   - 反常识角色设定的心理学原理
   - 极端约束激发创造力的案例
   - 结构化输出的工程化实践
4. 对比测试：Gemini vs GPT-4 vs Claude
5. 实战建议：如何为自己定制提示词库
```

### 4. 热点结合策略（Trend Mix）

将原内容与时事热点、节日节点、社会现象结合，增强时效性和传播力。

**适用场景：**
- 原内容主题可与当前热点关联
- 有节日、行业事件等可蹭的热点
- 可借势提升内容的传播力

**推文示例：**
```
原文：副业成为职场人的plan B
热点结合：2026元旦，不立flag，只算账。
主业月薪2万，副业收入3千→焦虑
主业月薪2万,副业收入8千→从容
主业月薪2万,副业收入2万→自由
新年第一课：plan B不是选项，是必需品。
```

**公众号大纲示例：**
```
标题：《2026年第一个工作日，给你的副业算笔账》
1. 开篇：为什么越来越多人在元旦规划副业
2. 三个真实案例：副业收入的三个台阶
   - 补贴级（3k-5k）：降低焦虑
   - 平衡级（8k-12k）：增加选择权
   - 自由级（>20k）：实现职业自主
3. 2026年最值得做的5个副业方向
4. 从0到1启动副业的30天计划
5. 风险提示：副业的三大误区
6. 新年行动清单
```

## 选题策略组合

**高传播组合：**
- 反驳 + 热点结合（引发讨论+蹭热度）
- 扩充 + 热点结合（干货+时效性）

**深度价值组合：**
- 延伸 + 扩充（全面+深入）
- 扩充 + 反驳（正反对比+系统方法）

**快速产出组合：**
- 延伸（基于原内容改写）
- 热点结合（借势创作）
